## 🎬 これから起きること

次に Claude Code に **「`git pull origin main` で最新を取り込んで」** と頼むと、以下のものが一気に降ってきます:

| 降ってくるもの | 中身 |
|---|---|
| 📁 `my-profile/` | **Stage 1〜5 で作るはずだったプロフィールページの完成形コード一式** |

⭐ **これが `git pull` の正体です。** 「サーバ側にだけある新しいコミットが、まるごとあなたのパソコンに降りてくる」── 仕事でも毎日この往復が起きます。

## ❌ うまくいかないとき

| 症状 | やること |
|---|---|
| 「Already up to date（もう最新です）」と出る | start タグではなく最新を clone してしまった可能性。**いったん `pull-practice` フォルダを消して、Claude Code に「`git clone -b start` でやり直して」と頼む** |
| 「You are not currently on a branch」 | `git checkout main` を頼んでから再度 pull |
| 違うフォルダで pull している | `pull-practice` フォルダ**の中**にいるか確認。Claude Code に「今いるフォルダを教えて」と聞く |
