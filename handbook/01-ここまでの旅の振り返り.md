# 🎒 01 — ここまでの旅の振り返り

Stage 1〜5、あなたは **5つの大きな山**を越えてきました。1つずつ、何を体験したか整理します。

---

## 🏔 Stage 1 — Antigravity のインストールと環境構築

**やったこと:** Antigravity・GitHub・Supabase・Render の4つのツールを準備して、`claude1` で Claude Code を起動した

⭐ **身についた感覚:** 「AI に頼める準備が整った」という土台

「ターミナル」「コマンドを打つ」という、今まで縁のなかった世界に足を踏み入れた瞬間です。

---

## 🏔 Stage 2 — Antigravity の画面に慣れる

**やったこと:** ファイル作成・編集・ターミナルで Claude Code 起動の3操作を覚えた

⭐ **身についた感覚:** 「画面のどこに何があるか」が分かる感覚

エディタは「コードを書く場所」ではなく **「AIに指示する場所」** だと体感したはず。

---

## 🏔 Stage 3 — Claude Code で何ができるのか？

**やったこと:** 曖昧プロンプト vs 具体的プロンプト の違いを目で見て体感した

⭐ **身についた感覚:** **「言葉に落とすほどAIが思い通りに動く」** という、AI時代の最重要スキルの入口

> ※ Stage 3 で「言語化の精度」を受講

---

## 🏔 Stage 4 — 自己紹介ページを作る

**やったこと:** Vite + React + TypeScript + Tailwind CSS + shadcn/ui で `my-profile` プロジェクトを作り、Card で自己紹介ページを表示した

⭐ **身についた感覚:** **「AIに頼んだだけで自分のページがブラウザに動いた」** という、AI時代の働き方の最初の実感

このとき出てきた英語（Vite, React, Tailwind, shadcn/ui...）は **覚えなくてOK**。後で `05-用語ミニ辞典.md` を見れば分かります。

---

## 🏔 Stage 5 — GitHubに保存して取り出してみる ⬅ いまここ

**やっていること:**
1. ✅ 自分の `my-profile` を **GitHub に push**（保存）
2. ✅ 講座の公開リポジトリ（このリポジトリ）を **`-b start` で clone**
3. ✅ **`git pull origin main`** で最新を取り込む ⬅ pull したからこれを読めている！
4. ⏳ 残り: 2枚のスクショと GitHubリポジトリURLを提出

⭐ **身についた感覚:**
- 「自分のコードを世界中どこからでも取り出せる場所に置いた」
- 「他の人の更新を、自分のパソコンに降ろせる」── これが **チーム開発の最小単位**
