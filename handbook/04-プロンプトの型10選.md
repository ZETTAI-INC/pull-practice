# 💬 04 — プロンプトの型 10選

Claude Code に何と頼めばいいか **「型」さえ覚えればいい**。10個だけ、コピペで使える型を集めました。

⭐ **使い方:** プロンプトの `[XXX]` 部分を、自分の状況に置き換えてコピペするだけ。

---

## 🔧 1. エラー対処の鉄板型

```
このエラーを直して:
[エラーメッセージを全文コピペ]
```

⭐ **効くシーン:** 画面が真っ白・赤い文字が出た・コマンドが失敗した、全部に効く。

> ※ Stage 4 のエラー対処で受講した「DevTools → Console → 赤い文字をコピー → AI に貼る」の流れ

---

## 🆘 2. 「何が起きているか分からない」型

```
今のターミナルの出力をそのまま貼ります。
[ターミナルの最後20行をコピペ]
これは何が起きているか、初心者向けの言葉で教えて。
次に私が何をすればいいかも提案して。
```

⭐ **効くシーン:** 英語のメッセージがたくさん出て、何が悪いのか分からない時。

---

## 🏗 3. プロジェクト作成型

```
今 開いているフォルダに [技術スタック] のプロジェクトをセットアップして。
[ライブラリ1] [ライブラリ2] も入れて。
npm install まで完了させて。完了したら次に何を確認すればいいか教えて。
```

例: `Vite + React + TypeScript + Tailwind CSS + shadcn/ui のプロジェクトを...`

> ※ Stage 4 のプロジェクト作成で受講した型

---

## 🎨 4. UI作成型（具体指示）

```
[画面名] を作って。
- 使う部品: [shadcn/ui の Card / Button など]
- 項目: [項目1・項目2・項目3]
- 見た目: [中央配置 / 青系テーマ / カード形式]
- データ: サンプルでOK
```

⭐ **効くシーン:** 自己紹介ページ・ダッシュボード・LP など、新しい画面を作りたい時。

> ※ Stage 3 で「具体的に書く」を受講

---

## 📚 5. 説明お願い型

```
[コードのファイル名] を開いて、何をしているか初心者向けの言葉で1段落ずつ教えて。
専門用語が出たら、その都度カッコで例えを添えて。
```

⭐ **効くシーン:** お手本コードを読むとき・他人のコードを理解したいとき。

---

## 🪞 6. 見比べ型

```
私の [ファイルA] と [ファイルB] を見比べて、
大きな違いを3つだけ初心者向けに教えて。
正解・不正解ではなく「こういう書き方もある」という発見として。
```

⭐ **効くシーン:** お手本と自分のコードを比べたい時。

---

## 💾 7. GitHub に保存型

```
今の変更をコミットして GitHub に push して。
コミットメッセージは「[何をしたか短く]」で。
.gitignore に .env* が含まれているかも確認して。
```

> ※ Stage 5 で受講

---

## 📥 8. 取り込み型（clone / pull）

```
このフォルダで git pull origin main を実行して、
新しく追加されたファイル名と、それぞれの中身を2〜3行で短く教えて。
```

⭐ **効くシーン:** 他の人がアップした更新を取り込みたい時。

> ※ Stage 5 で受講（このリポジトリを取り込んだ時に使ったプロンプト）

---

## 🔍 9. レビュー依頼型

```
私が書いた [ファイル名] を、初心者が次に成長するための観点でレビューして。
直さなくていい、コメントだけで。
「これは良い」「これはこうした方がもっと良くなる」を3個ずつ。
```

⭐ **効くシーン:** 自分のコードに自信がない時。AI に **コーチ役**になってもらえる。

---

## 🆘 10. 「白紙に戻したい」型

```
このプロジェクトが今 どんな状態か分からなくなった。
- フォルダ構成を教えて
- 何のライブラリが入っているか教えて
- 今 ブラウザで何が表示されるはずか教えて
- 次に何をすればいいか提案して
```

⭐ **効くシーン:** 数日空けて戻ってきた時・何をしていたか忘れた時。

---

## 🎁 おまけ: プロンプトの「3点セット」

困ったら、この3点だけ意識すれば伝わります:

| # | 何を書く | 例 |
|---|---|---|
| 1️⃣ | **何をしたいか** | 「自己紹介ページを作りたい」 |
| 2️⃣ | **どんな見た目** | 「Card で中央配置、青系」 |
| 3️⃣ | **どこに困っているか** | 「画面が真っ白で、Consoleにこのエラーが出ている」 |

⭐ この3点が揃えば、AI はあなたを助けられます。
